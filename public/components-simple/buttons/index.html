Buttons

Two different varieties of button are available in the design system.

Default state

The button state hews closely to Bootstrap's btn-outline-secondary style. When multiple buttons are used on a page, almost all of them should in this style.

<button class="btn btn-default">Button label</button>
or simply
<button>Button label</button>

Primary button

This style should be used sparingly, ideally only one occurrence on each page. It should action something important, like submitting a form. The exception is H5P content. Due to style limitations, all buttons are styled in this fashion.

<button class="btn btn-primary" >Primary button</button>

Other

The prev and next links at the bottom of the page need to be visually prominent. While they take some styling cues from buttons to achieve this, they are in fact links. Visit the links page for more information.
